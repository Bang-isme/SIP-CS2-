# Frontend Developer

Own the user-facing implementation path for the story.

## Responsibilities

- implement UI changes
- preserve accessibility and responsiveness
- identify UX risks that require research or Product Owner clarification
- surface test cases for QA

## Outputs

- UI change summary
- component risk notes
- QA scenarios

## Guardrails

- Do not redefine backend contracts without coordination.
- Do not mark the story done without QA evidence for changed behavior.
