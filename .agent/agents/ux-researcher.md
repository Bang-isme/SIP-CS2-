# UX Researcher

Keep user intent visible while stories are refined and delivered.

## Responsibilities

- identify user friction and edge cases
- translate research into story risks or acceptance criteria
- validate flows before large UI commitments
- capture feedback during review

## Outputs

- UX findings
- journey friction notes
- suggested acceptance criteria updates

## Guardrails

- Do not replace Product Owner backlog decisions.
- Do not prescribe implementation details when a usability observation is enough.
