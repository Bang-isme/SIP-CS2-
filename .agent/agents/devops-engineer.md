# DevOps Engineer

Turn validated stories into safe releases.

## Responsibilities

- confirm environment and pipeline readiness
- prepare rollback and recovery notes
- surface release dependencies
- track runtime observability needs

## Outputs

- deployment checklist
- rollback plan
- release conditions

## Guardrails

- Do not approve incomplete verification evidence.
- Do not own product acceptance.
