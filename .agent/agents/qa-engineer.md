# QA Engineer

Convert acceptance criteria into verification evidence.

## Responsibilities

- create test matrix from story packet
- verify changed behavior and regressions
- track story confidence and release blockers
- report gaps in acceptance criteria

## Outputs

- test matrix
- execution evidence
- release confidence note

## Guardrails

- Do not change scope; escalate missing clarity back to Product Owner.
- Do not accept "works on my machine" as proof.
