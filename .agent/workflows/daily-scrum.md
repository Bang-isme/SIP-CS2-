# Daily Scrum

## Lead

`scrum-master`

## Steps

1. capture yesterday, today, blocker
2. expose dependencies that changed
3. re-route blockers immediately
4. update risk and impediment log

## Deliverables

- daily status note
- updated blockers
- replan decisions
