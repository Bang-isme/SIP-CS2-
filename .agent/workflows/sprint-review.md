# Sprint Review

## Lead

`product-owner`

## Support

`scrum-master`, delivery leads, `qa-engineer`, optional `ux-researcher`

## Steps

1. demo tested increments only
2. capture stakeholder feedback
3. separate accepted work from future requests
4. update backlog and release notes

## Deliverables

- accepted increment list
- feedback backlog
- follow-up actions
