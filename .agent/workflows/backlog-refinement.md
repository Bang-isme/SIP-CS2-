# Backlog Refinement

## Lead

`product-owner`

## Support

`scrum-master`, `solution-architect`, optional `ux-researcher`

## Steps

1. capture user value and desired outcome
2. split scope into story-sized increments
3. define acceptance criteria and verification hints
4. note dependencies, risks, and open questions

## Deliverables

- prioritized story packet
- dependency notes
- ready/not-ready decision
